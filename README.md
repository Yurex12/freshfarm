# FreshFarm

A simple 3-page farm website built with React, TypeScript, and Tailwind CSS.
The website includes Home, About, and Contact pages and is fully responsive.

## Live Website

https://freshfarm.vercel.app

## Pages

- Home
- About
- Contact

## Tech Stack

- React
- TypeScript
- Tailwind CSS

## Deployment

Hosted on Vercel.
